# Multiplayer Game Online

Sistem game multiplayer online dengan ID room acak 6 angka.

## Fitur Utama

- ✅ Buat room dengan ID acak 6 digit
- ✅ Invite pemain dengan membagikan ID room
- ✅ Multiplayer real-time dengan Socket.IO
- ✅ Sistem lobi dengan status "Ready"
- ✅ Game Tic-Tac-Toe (contoh implementasi)
- ✅ Chat dalam game
- ✅ Sistem giliran (turn-based)
- ✅ Skor dan pemenang
- ✅ Responsive design

## Teknologi

- **Backend**: Node.js, Express, Socket.IO
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: In-memory (Map) untuk penyimpanan sederhana
